About MyLibrary

MyLibrary is a digital library management system designed to simplify the process of managing books and user interactions in a library environment. The system provides an efficient and user-friendly way to browse, issue, and track books using modern web technologies.

Project Overview

MyLibrary is developed as a frontend-only web application using HTML, CSS, and JavaScript. Instead of a backend database, the application uses localStorage to store user data, book information, and issued book records. This makes the system lightweight, fast, and easy to deploy.

Objectives

  • Provide a simple and intuitive interface for library users
  • Allow users to register and log in securely
  • Enable users to browse and search available books
  • Manage book issue and return functionality
  • Maintain records of issued books

Technologies Used

  • HTML – Page structure and content
  • CSS – Styling, layout, and responsiveness
  • JavaScript – Interactivity, validation, and application logic
  • localStorage – Client-side data storage

Library Rules and Policies

  • Only registered users are allowed to issue books
  • A book cannot be issued if it is already marked as issued
  • Books must be returned before issuing new ones
  • All issue and return actions are recorded automatically

Benefits

  • Reduces manual record keeping
  • Provides quick access to book availability
  • Ensures accurate tracking of issued books
  • Requires no backend or server setup
